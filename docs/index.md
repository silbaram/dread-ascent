# Dread Ascent

웹 브라우저에서 즐기는 **2D 턴제 로그라이트 던전 탐험 게임**.

플레이어는 층수가 올라갈수록 강력해지는 던전을 탐험하며 아이템을 파밍하고 적을 처치합니다. 사망해도 영구 성장 요소를 통해 다음 도전을 유리하게 이어갑니다. 최종 목표는 100층의 최종 보스를 물리치는 것입니다.
