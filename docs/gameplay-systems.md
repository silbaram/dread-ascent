# Gameplay Systems

## 코어 루프

현재 게임 루프는 아래 흐름으로 이해하면 됩니다.

1. 타이틀 화면 진입
2. 새 런 시작 또는 저장된 런 이어하기
3. 층 생성과 탐험
4. 적 조우 시 카드 전투
5. 적 처치 후 카드 보상 오퍼 가능
6. 계단으로 다음 층 이동
7. 사망 시 게임 오버와 영혼 파편 정산
8. 타이틀로 돌아가 성소/카드 모음집 사용

## 층 진행

층 타입은 세 가지입니다.

- `normal`
  일반 탐험 층
- `safe`
  적이 없는 안전지대 층
- `boss`
  보스층

현재 규칙:

- 1층은 항상 `normal`
- 다음 층은 25% 확률로 `safe`
- 100층 이상은 `boss`

## 탐험

탐험은 `MainScene`이 담당합니다.

- 플레이어는 격자 단위로 이동합니다.
- 시야는 FOV 기반으로 갱신됩니다.
- 적은 자신의 턴에 시야와 마지막 목격 위치를 기준으로 이동/공격합니다.
- 아이템 위로 이동하면 자동으로 주울 수 있습니다.
- 계단에 올라가면 다음 층으로 진행합니다.

안전지대(`safe`)는 현재 다음 성격을 가집니다.

- 적이 없습니다.
- `REST` 타일과 계단이 배치됩니다.
- `REST` 타일에 올라가면 로그/턴 진행은 있지만, 전용 회복 시스템은 아직 없습니다.

## 전투

전투는 `BattleScene`으로 분리되어 있습니다.
적과 조우하면 덱이 비어 있어도 같은 `BattleScene` 전투 모델로 들어갑니다.

### 플레이어 턴

- 턴 시작 시 에너지가 최대치로 리필됩니다.
- 기본 드로우는 5장입니다.
- 손패 카드를 클릭해 사용합니다.
- 에너지가 0이 되면 자동 턴 종료가 일어날 수 있습니다.

### 카드 효과

카드 처리는 `CardEffectService`가 맡습니다.

- `DAMAGE`
- `BLOCK`
- `STATUS_EFFECT`
- `DRAW`
- `HEAL`
- `MULTI_HIT`
- `DAMAGE_BLOCK`
- `BUFF`
- `DISCARD_EFFECT`
- `CONDITIONAL`
- `FLEE`

### Block 규칙

- 공격 피해는 대상의 `block`을 먼저 소모합니다.
- 남은 피해만 체력에 들어갑니다.
- 유지 버프가 없으면 Block은 소유자의 다음 턴 시작 시 정리됩니다.
- `Barricade`, `Brace` 같은 카드가 Block 유지 규칙을 바꿀 수 있습니다.

### 상태이상

주요 상태이상:

- `VULNERABLE`
  받는 피해 증가
- `WEAK`
  주는 피해 감소
- `POISON`
  턴 종료 시 피해
- `FRAIL`
  Block 효율 감소
- `STRENGTH`
  공격 보정
- `REGENERATION`
  턴 종료 회복
- `THORNS`
  피격 반사

### 적 intent

적은 매 턴 다음 행동을 미리 보여줍니다.

- `Next ⚔️ n`
- `Next 🛡️ +n`
- `Next ⬆️ ATK +n`

이 intent는 "현재 수치"가 아니라 "다음 턴 예정 행동"입니다.

## 보상

적 처치 후 카드 보상은 `CardDropService`가 판정합니다.

- 일반 적: 30%
- 엘리트 적: 60%
- 보스: 100%

드롭이 성공하면 3장 오퍼를 생성합니다.

- 덱 방향성과 맞는 카드 1장
- 중립 카드 1장
- 랜덤 카드 1장

덱이 가득 차 있으면 교체 플로우가 열릴 수 있습니다.

## 덱과 카드 빌드

시작 덱:

- `Strike x4`
- `Fortify x3`

현재 카드 축:

- `NEUTRAL`
- `BLOOD_OATH`
- `SHADOW_ARTS`
- `IRON_WILL`
- `CURSE`

대표 빌드 감각:

- `BLOOD_OATH`: 자해와 저체력 페이오프
- `SHADOW_ARTS`: 독과 디버프 누적
- `IRON_WILL`: Block 축적과 방어 기반 공격

## 진행과 메타

런 중 성장:

- 적 처치 시 경험치 획득
- 레벨업 시 기본 스탯 상승

런 바깥 성장:

- 게임 오버 시 영혼 파편 지급
- 타이틀 화면 성소에서 영구 업그레이드 구매
- 이전 하강에서 확보한 카드는 카드 모음집에 누적

## 현재 코드 기준으로 알아둘 점

- 영혼 파편은 성소에서 얻는 것이 아니라 게임 오버 시 정산됩니다.
- 승리 화면에서는 현재 영혼 파편 지급 로직이 없습니다.
- 안전지대는 존재하지만, 현재 회복 장치 시스템은 구현되어 있지 않습니다.
