상태이상(독, 취약, 약화, 허약)과 유지 버프(가시, 방어도 유지)의 구체적인 작동 방식을 괄호 안에 명시하여, 앞서 논의된 모든 개선안을 반영한 최종 텍스트 버전입니다.

---

### 1. 중립 (NEUTRAL)
기본적인 공격, 방어, 유틸리티를 담당하는 카드군입니다.

* **Strike (타격)**
    * **능력치:** 비용 1 / 피해량 6
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 단일 대상에게 6의 피해를 줍니다.

* **Fortify (요새화)**
    * **능력치:** 비용 1 / 방어도 5
    * **분류:** 일반(Common) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 자신에게 5의 방어도를 부여합니다.

* **Quick Draw (빠른 뽑기)**
    * **능력치:** 비용 1 / (유틸리티)
    * **분류:** 일반(Common) / 스킬(SKILL)
    * **키워드:** 없음
    * **설명:** 덱에서 카드를 2장 뽑습니다.

* **Heavy Strike (강타)**
    * **능력치:** 비용 2 / 피해량 12
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 단일 대상에게 12의 피해를 줍니다.

* **Shockwave (충격파) ✨ [광역기 변경]**
    * **능력치:** 비용 2 / 피해량 8
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 전투에 참여 중인 **모든 적**에게 각각 8의 피해를 줍니다. (적의 수가 많을수록 효율이 극대화됩니다.)

* **Shadow Step (그림자 걸음)**
    * **능력치:** 비용 0 / (유틸리티)
    * **분류:** 희귀(Rare) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 전투에서 도주(FLEE)합니다.

* **Adrenaline (아드레날린)**
    * **능력치:** 비용 0 / (유틸리티)
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 카드를 1장 뽑고 에너지를 1 얻습니다.

* **Recycle (재활용)**
    * **능력치:** 비용 0 / (유틸리티)
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 없음
    * **설명:** 카드를 1장 버리고, 2장을 뽑습니다.

* **Second Wind (재충전)**
    * **능력치:** 비용 1 / 회복량 6
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 체력을 6 회복합니다.

### 2. 피의 맹세 (BLOOD_OATH) ✨ 리워크 2.0 적용
자해와 저체력 상태일 때 폭발적인 이득을 얻는 하이 리스크 하이 리턴 빌드의 카드들입니다.

* **Blood Price (피의 대가)**
    * **능력치:** 비용 0 / (유틸리티)
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 없음
    * **설명:** 자신에게 4의 피해를 입히고 카드를 2장 뽑습니다.

* **Adrenaline Rush (아드레날린 쇄도)**
    * **능력치:** 비용 1 / (유틸리티)
    * **분류:** 희귀(Rare) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 자신에게 5의 피해를 입힙니다. 에너지를 3 얻고 카드를 3장 뽑습니다.

* **Crimson Pact (진홍빛 계약)**
    * **능력치:** 비용 1 / (버프)
    * **분류:** 고급(Uncommon) / 파워(POWER)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 자신에게 5의 피해를 입히고, 이번 전투 동안 유지되는 근력(STRENGTH)을 2 얻습니다.

* **Blood Shield (피의 방패)**
    * **능력치:** 비용 1 / 방어도 12
    * **분류:** 고급(Uncommon) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 자신에게 3의 피해를 입히고 방어도를 12 얻습니다.

* **Death Wish (죽음의 소원)**
    * **능력치:** 비용 2 / 피해량 (조건부 가변)
    * **분류:** 희귀(Rare) / 공격(ATTACK)
    * **키워드:** 보존(RETAIN)
    * **설명:** 내가 잃은 체력 1당 1.5의 피해를 줍니다.

* **Reckless Fury (무모한 분노)**
    * **능력치:** 비용 1 / 피해량 3 (4번 타격)
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 자신에게 2의 피해를 입히고, 대상에게 3의 피해를 4번(다단 히트) 줍니다.

* **Bloodrush (피의 쇄도)**
    * **능력치:** 비용 2 / 피해량 20
    * **분류:** 고급(Uncommon) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 20의 피해를 줍니다. 내 체력이 50% 이하라면 비용이 0이 됩니다.

* **Last Stand (최후의 항전)**
    * **능력치:** 비용 3 / 피해량 40
    * **분류:** 희귀(Rare) / 공격(ATTACK)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 40의 피해를 줍니다. 이 카드로 적을 처치하면 최대 체력의 30%를 회복합니다. 내 체력이 25% 이하라면 비용이 0이 됩니다.

* **Berserker Rage (광전사의 분노)**
    * **능력치:** 비용 2 / (패시브 버프)
    * **분류:** 희귀(Rare) / 파워(POWER)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 이번 전투 동안, 적에게 맞거나 자해를 할 때마다 근력(STRENGTH)을 1 얻습니다.

### 3. 그림자 술법 (SHADOW_ARTS) ✨ 상태이상 메커니즘 보완
독 상태이상과 디버프를 누적시켜 전투를 유리하게 이끄는 카드군입니다.

* **Venom Strike (맹독 타격)**
    * **능력치:** 비용 1 / 피해량 4
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 4의 피해를 줍니다. 대상에게 **독(POISON)을 3 스택** 부여합니다. *(독에 걸린 대상은 턴이 끝날 때마다 독 스택에 비례한 피해를 입으며, 이후 스택이 1씩 감소합니다.)*

* **Weaken (약화)**
    * **능력치:** 비용 1 / (디버프)
    * **분류:** 일반(Common) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 대상에게 2턴 지속되는 **취약(VULNERABLE)**을 부여합니다. *(취약 상태인 대상은 공격으로 받는 피해량이 일정 비율 증가합니다.)*

* **Miasma (독기)**
    * **능력치:** 비용 1 / (디버프)
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 대상에게 **독(POISON)을 5 스택** 부여합니다. *(스택에 비례한 턴 종료 피해 발생 및 스택 감소)*

* **Smoke Screen (연막)**
    * **능력치:** 비용 1 / 방어도 4
    * **분류:** 고급(Uncommon) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 방어도 4를 얻습니다. 대상에게 1턴 지속되는 **약화(WEAK)**를 부여합니다. *(약화 상태인 대상은 공격 시 주는 피해량이 일정 비율 감소합니다.)*

* **Shadow Cloak (그림자 망토)**
    * **능력치:** 비용 1 / 방어도 6
    * **분류:** 고급(Uncommon) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 방어도 6을 얻고 카드를 1장 뽑습니다.

* **Exploit Weakness (약점 노출)**
    * **능력치:** 비용 1 / 피해량 (조건부 가변)
    * **분류:** 고급(Uncommon) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 대상이 가진 디버프 수 1당 4배수로 비례하는 피해를 줍니다.

* **Crippling Blow (치명타)**
    * **능력치:** 비용 2 / 피해량 8
    * **분류:** 고급(Uncommon) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 8의 피해를 줍니다. 대상에게 1턴 지속되는 **약화(WEAK)**와 **허약(FRAIL)**을 모두 부여합니다. *(허약 상태인 대상은 카드 사용으로 얻는 방어도가 일정 비율 감소합니다.)*

* **Toxic Burst (맹독 폭발)**
    * **능력치:** 비용 1 / (디버프 증폭)
    * **분류:** 희귀(Rare) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 대상이 현재 가지고 있는 **독(POISON) 스택을 2배로 증폭**시킵니다.

* **Noxious Aura (유독성 오라)**
    * **능력치:** 비용 2 / (패시브 버프)
    * **분류:** 희귀(Rare) / 파워(POWER)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 매 턴 대상에게 독 2를 부여하는(APPLY_POISON_PER_TURN) 오라 효과를 얻습니다.

### 4. 강철의 의지 (IRON_WILL) ✨ 방어 버프 메커니즘 보완
방어도를 견고히 쌓고, 그 방어도를 활용해 전투를 보조하거나 공격을 수행하는 방어 특화 카드군입니다.

* **Iron Guard (강철 호위)**
    * **능력치:** 비용 2 / 방어도 10
    * **분류:** 일반(Common) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 자신에게 10의 방어도를 부여합니다.

* **Shield Bash (방패 타격)**
    * **능력치:** 비용 2 / 피해량 (조건부 가변)
    * **분류:** 고급(Uncommon) / 공격(ATTACK)
    * **키워드:** 없음
    * **설명:** 현재 자신이 가진 방어도 1당 1비율로 스케일링된 데미지를 줍니다.

* **Reinforce (강화)**
    * **능력치:** 비용 1 / 방어도 5
    * **분류:** 고급(Uncommon) / 방어(GUARD)
    * **키워드:** 없음
    * **설명:** 방어도 5를 얻습니다. 자신에게 2턴 지속되는 **가시(THORNS)**를 2 스택 부여합니다. *(가시 상태일 때 공격을 받으면, 공격한 적에게 스택만큼의 반사 피해를 줍니다.)*

* **Brace (대비)**
    * **능력치:** 비용 1 / 방어도 4
    * **분류:** 고급(Uncommon) / 방어(GUARD)
    * **키워드:** 보존(RETAIN)
    * **설명:** 방어도 4를 얻습니다. 이번 턴에 얻은 방어도가 **턴 종료 시 사라지지 않고 다음 턴까지 유지(BLOCK_PERSIST)**됩니다. *(기본적으로 남은 방어도는 턴 종료 시 모두 소멸합니다.)*

* **Counter Strike (반격)**
    * **능력치:** 비용 1 / 피해량 (조건부 가변)
    * **분류:** 고급(Uncommon) / 공격(ATTACK)
    * **키워드:** 보존(RETAIN)
    * **설명:** 이번 턴에 피해를 최소 1 이상 받았을 때 조건부로 발동하며, 이번 턴에 받은 피해량에 비례하여(1:1 스케일링) 적에게 데미지를 줍니다.

* **Taunt (도발)**
    * **능력치:** 비용 0 / (디버프)
    * **분류:** 고급(Uncommon) / 스킬(SKILL)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** 대상의 공격력을 1턴 동안 3 감소시킵니다.

* **Barricade (바리케이드)**
    * **능력치:** 비용 3 / (버프)
    * **분류:** 희귀(Rare) / 파워(POWER)
    * **키워드:** 소멸(EXHAUST)
    * **설명:** **이번 전투 동안**, 턴이 종료되어도 남은 방어도가 사라지지 않고 계속 누적 유지(BLOCK_PERSIST)됩니다.

### 5. 저주 (CURSE)
플레이어에게 패널티를 주는 카드들입니다.

* **Hemorrhage (출혈)**
    * **능력치:** 비용 0 / (패널티)
    * **분류:** 희귀(Rare) / 저주(CURSE)
    * **키워드:** 플레이 불가(UNPLAYABLE)
    * **설명:** 플레이할 수 없으며, 손에 있을 시 조건부로 1의 자해 피해를 줍니다.

* **Dread (공포)**
    * **능력치:** 비용 0 / (패널티)
    * **분류:** 희귀(Rare) / 저주(CURSE)
    * **키워드:** 플레이 불가(UNPLAYABLE), 휘발성(ETHEREAL)
    * **설명:** 플레이할 수 없으며 턴이 끝나면 소멸합니다.
