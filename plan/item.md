> 등급 체계: COMMON / UNCOMMON / RARE / EPIC / CURSED (확정)
> 장비 슬롯: 무기(WEAPON), 투구(HELMET), 갑옷(BODY_ARMOR), 신발(BOOTS)
> 아키타입: 피의 맹세(Blood Oath), 그림자 예술(Shadow Arts), 철의 의지(Iron Will)

## 1. 밸런스 기준 — 왜 이 숫자인가

### 1-1. 현재 게임의 핵심 수치

| 항목               | 값     | 출처                    |
| ------------------ | ------ | ----------------------- |
| 플레이어 기본 ATK  | 10     | BASE_PLAYER_STATS       |
| 플레이어 기본 DEF  | 5      | BASE_PLAYER_STATS       |
| 플레이어 기본 HP   | 100    | BASE_PLAYER_STATS       |
| 적 기본 ATK/DEF    | 10 / 5 | BASE_ENEMY_STATS        |
| 적 층당 HP 증가    | +10    | ENEMY_HEALTH_PER_FLOOR  |
| 적 층당 ATK 증가   | +1     | ENEMY_ATTACK_PER_FLOOR  |
| 적 층당 DEF 증가   | +1     | ENEMY_DEFENSE_PER_FLOOR |
| Strike 기본 데미지 | 6      | CARD_BALANCE_TABLE      |
| Fortify 기본 Block | 5      | CARD_BALANCE_TABLE      |
| 에너지 / 턴        | 3      | COMBAT_RESOURCE_BALANCE |

### 1-2. 장비 → 카드 파워 변환 구조

`EquipmentCardBonusService`가 장비 스탯을 카드에 직접 가산한다.

```
무기 ATK +3 → 모든 공격 카드에 +3 데미지 (Strike 6→9, Bloodrush 18→21)
방어구 DEF +2 → 모든 수비 카드에 +2 Block (Fortify 5→7)
```

이 구조 때문에 **장비 스탯 1당 가치가 매우 높다**. 턴당 공격 카드를 2~3장 쓴다고 하면, 무기 ATK +3은 실제로 턴당 +6~9 데미지에 해당한다. 그래서 장비 스탯은 작은 숫자여야 한다.

### 1-3. 장비 슬롯 확장에 따른 밸런스 위험

기존: WEAPON(1) + ARMOR(1) = 장비 2칸
변경: WEAPON(1) + HELMET(1) + BODY_ARMOR(1) + BOOTS(1) = 장비 4칸

**방어 슬롯이 1→3으로 늘어남.** 각 방어 장비의 DEF를 기존 ARMOR와 동일하게 주면 총 DEF가 3배가 되어 밸런스가 붕괴된다.

### 1-4. 해결: 슬롯별 스탯 예산 분배

| 슬롯       | 주 역할       | DEF 예산 비중 | 설계 의도                          |
| ---------- | ------------- | ------------- | ---------------------------------- |
| HELMET     | 인지/유틸리티 | 20%           | FOV, 카드 드로우, 상태이상 저항 등 |
| BODY_ARMOR | 방어 핵심     | 60%           | 주요 DEF 담당                      |
| BOOTS      | 템포/행동력   | 20%           | 에너지, 선공, 도주 확률 등         |

COMMON 풀셋 기준으로 기존 ARMOR DEF +2와 **동일한 총합**이 되도록 조정:

| 슬롯       | COMMON DEF | UNCOMMON DEF | RARE DEF | EPIC DEF |
| ---------- | ---------- | ------------ | -------- | -------- |
| HELMET     | 0          | 0~1          | 1        | 1        |
| BODY_ARMOR | 2          | 3            | 4        | 5        |
| BOOTS      | 0          | 0~1          | 1        | 1        |
| **합계**   | **2**      | **3~5**      | **6**    | **7**    |

무기 ATK는 기존과 동일한 스케일 유지:

| 등급     | 무기 ATK |
| -------- | -------- |
| COMMON   | +2~3     |
| UNCOMMON | +3~4     |
| RARE     | +5~6     |
| EPIC     | +6~7     |

## 3. 투구 (HELMET) — 10종

### 설계 철학
DEF 기여는 미미하거나 없음. 대신 **인지·전략 보조** 효과에 집중한다. 카드 드로우, 적 의도 관련, 상태이상 저항, 전투 시작 유틸리티 등. "머리를 쓰는 것"에 관한 장비.

---

### COMMON (2종)

**H-01. Leather Cap (가죽 모자)**
- DEF +0, MaxHP +5
- 고유 효과: 없음
- 설명: 약간의 추가 체력을 제공하는 기본 투구.

**H-02. Iron Helm (철제 투구)**
- DEF +1
- 고유 효과: 없음
- 설명: 단단한 철 투구. 머리를 보호한다.

---

### UNCOMMON (3종)

**H-03. Scout's Visor (정찰 바이저)**
- DEF +0, MaxHP +5
- 고유 효과: 전투 시작 시 카드 1장 추가 드로우 (첫 턴만)
- 아키타입: 범용
- 밸런스 근거: 첫 턴 6장 → 선택지 넓어짐. 1회성이라 Slay the Spire의 Bag of Preparation(렐릭)보다 약함. UNCOMMON 투구로 적절.

**H-04. Blood Crown (피의 관)**
- DEF +0
- 고유 효과: 자기 HP가 감소할 때마다 다음 공격 카드 파워 +2 (1회, 소모됨)
- 아키타입: 피의 맹세
- 밸런스 근거: Blood Fang 무기와 조합 시 자해 1회당 추가 데미지 +2 적립. 적립은 다음 공격에 1회만 소비되므로 무한 스케일링 아님. 턴당 1~2회 자해 기준 +2~4 추가 데미지.

**H-05. Watcher's Eye (감시자의 눈)**
- DEF +1
- 고유 효과: 적의 의도가 "공격"일 때 해당 턴 수비 카드 Block +2
- 아키타입: 철의 의지
- 밸런스 근거: 적 공격 턴에만 발동하므로 약 50~60% 턴에서 효과. Fortify 5→7, Iron Wall 10→12. 적 의도 시스템(EnemyIntentService)과 직접 시너지.

---

### RARE (2종)

**H-06. Plague Doctor's Mask (역병 의사의 가면)**
- DEF +1
- 고유 효과: 적에게 부여한 Poison이 턴당 1 더 감소하지 않음 (감소량 1→0, 즉 Poison이 안 줄어듦)
- 아키타입: 그림자 예술
- 밸런스 근거: 현재 Poison은 STATUS_EFFECT_BALANCE.poison.stackDecayPerTurn = 1로 매 턴 1씩 감소. 이 투구를 쓰면 Poison이 유지되어 누적 데미지가 비약적으로 증가. Poison 10 기준: 가면 없으면 총 55 데미지(10+9+8+...+1), 가면 있으면 10×N 턴 = 5턴에 50, 10턴에 100. 독 빌드의 핵심 장비.

**H-07. Tactician's Circlet (전술가의 관)**
- DEF +1
- 고유 효과: 턴 종료 시 손패에 남은 카드 1장당 다음 턴 카드 드로우 +1 (최대 +2)
- 아키타입: 범용
- 밸런스 근거: 에너지를 아끼고 카드를 남기면 다음 턴에 선택지가 넓어짐. 에너지 관리 보상. 최대 +2 캡으로 폭주 방지.

**H-08. All-Seeing Crown (전지의 왕관)**

- 밸런스 근거: 2턴 앞 예측이 가능하면 Block 축적 타이밍을 최적화할 수 있어 철의 의지 빌드에서 극강. 하지만 "정보"이지 직접 파워가 아니므로 숙련도에 따라 가치 차이 큼. 초보에게는 RARE급, 숙련자에게는 EPIC급.

---

### CURSED (2종)

**H-09. Runic Blindfold (룬의 눈가리개)**
- DEF +0
- 고유 효과(이점): 매 턴 카드 1장 추가 드로우 (5장→6장)
- 고유 효과(페널티): 적 의도 표시가 완전히 사라짐

**H-10. Madman's Hood (광인의 두건)**
- DEF +0
- 고유 효과(이점): 전투 시작 시 손패의 모든 카드 파워 +3
- 고유 효과(페널티): 전투 시작 시 자기에게 Vulnerable 2턴 부여
- 아키타입: 피의 맹세 시너지 (Vulnerable = 받는 데미지 ×1.5)
- 밸런스 근거: 첫 턴 공격 카드 2장 × +3 = +6 데미지. 대신 2턴간 받는 데미지 50% 증가. 공격 카드 2장(+6) + 방어 카드 1장의 Block은 변동 없으므로, 첫 2턴을 공격으로 밀어붙이고 빨리 끝내는 전략에 유리. Block 빌드에서는 Vulnerable 2턴이 치명적이라 선택 불가.

## 4. 갑옷 (BODY_ARMOR) — 10종

### 설계 철학
DEF의 주력 담당. 수비 카드(GUARD) 파워를 직접 올리는 핵심 슬롯. 등급이 올라갈수록 "맞을 때 반격" 또는 "Block 효율 증가" 같은 전투 중 방어 메커니즘이 추가된다.

---

**A-01. Leather Vest (가죽 조끼)**
- DEF +2
- 고유 효과: 없음
- 설명: 가볍고 기본적인 보호를 제공한다.

**A-02. Chain Mail (쇠사슬 갑옷)**
- DEF +2, MaxHP +5
- 고유 효과: 없음
- 설명: 촘촘한 고리가 베기 공격을 분산한다.

---

### UNCOMMON (3종)

## 8. EquipmentCardBonusService 변경 영향

현재 시스템은 WEAPON의 ATK → 공격 카드, ARMOR의 DEF → 수비 카드로 변환한다.

### 변경 후 규칙

```
WEAPON ATK → 공격 카드 파워 보너스 (기존과 동일)
HELMET DEF + BODY_ARMOR DEF + BOOTS DEF → 수비 카드 파워 보너스 (3슬롯 합산)
```

COMMON 풀셋 DEF 합산이 기존 ARMOR와 동일(+2~3)하므로 카드 보너스 밸런스 유지.

### 고유 효과 처리

고유 효과는 EquipmentCardBonusService가 아닌 BattleScene의 트리거 시스템에서 처리해야 한다. 기존 서비스는 순수 스탯 변환만 담당하고, 고유 효과는 별도의 `EquipmentEffectService`에서 관리하는 것이 아키텍처 원칙(도메인/렌더링 분리)에 부합.
